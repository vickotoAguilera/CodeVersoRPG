## 2. Editor Maestro de Animaciones (visualizador_recursos.py)

### Archivo Principal: `visualizador_recursos.py`
Lanzador: `visualizar_recursos.bat`

**Funcionalidades Implementadas:**

#### Sistema de Categorías y Archivos
- [x] Escaneo inteligente de carpetas en `assets/sprites`
- [x] Selector de carpeta de destino (categoría) que rota con click
- [x] Crear nuevas carpetas (+), escribir nombre, ENTER para confirmar
- [x] Eliminar carpetas con DEL/BACKSPACE (mientras hover en el selector)
- [x] Resaltado azul del archivo seleccionado en la lista lateral
- [x] Auto-selección de categoría si el recurso ya está en una carpeta conocida

#### Plantilla y Grid
- [x] Dibujar plantilla arrastrando sobre la imagen
- [x] Handle de redimensión en las 8 direcciones (esquinas + medios)
- [x] Generar grid con CORTAR TODO
- [x] **智能 Fill**: CORTAR TODO solo agrega en espacios VACÍOS, respeta frames ya posicionados
- [x] Botón LIMPIAR para empezar de nuevo

#### Manipulación de Frames
- [x] Crear frame nuevo: click en espacio vacío arrastrando (usa tamaño de plantilla)
- [x] Seleccionar: click, CTRL+click (agregar), caja de selección (arrastrar)
- [x] Mover grupo: click en frame ya seleccionado con varios activos
- [x] Redimensionar: 8 handles de control (esquinas + lados)
- [x] Eliminar: DEL o BACKSPACE (elimina el frame donde está el mouse)
- [x] Preview animado en el panel lateral

#### Guardado
- [x] Nombre del personaje + acción (ej: "marciano_caminando")
- [x] Guarda en `assets/sprites/[CATEGORÍA]/[PERSONAJE]/`
- [x] Numeración secuencial automática (1, 2, 3...)

### Controles del Editor
```
1. Arrastrar sobre la foto → Definir tamaño de plantilla
2. CLICK en espacio vacío → Crear frame con tamaño de plantilla
3. CORTAR TODO → Llena espacios vacíos (respeta frames existentes)
4. CLICK+ARRAS → Seleccionar zona
5. CTRL+CLICK → Agregar frame a selección
6. CLICK en ya selec → Mover grupo
7. Handles 8 dirs → Redimensionar frame
8. DEL/BACKSPACE → Eliminar frame
9. Click en selector categoría → Rotar carpetas
10. [+] + nombre + ENTER → Crear carpeta
11. Hover selector + DEL → Eliminar carpeta
```
