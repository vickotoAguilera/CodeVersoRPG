# Editor de Muros - Controles y Funciones

## Modos de Dibujo

- **R**: Modo Rectángulo
- **P**: Modo Pincel (dibujo continuo)
- **E**: Modo Borrador
- **L**: Modo Polígono libre (clicks para puntos, Enter para crear)

## Polígonos

- **Click izquierdo**: Agrega vértice
- **Click derecho**: Elimina el último vértice
- **Enter**: Finaliza y crea el muro poligonal (mínimo 3 puntos)
- **ESC**: Cancela el polígono en progreso

## Edición de Muros

- **Click izquierdo**: Selecciona muro
- **Shift + Click**: Multi-selección (agrega/quita de la selección)
- **Arrastrar**: Mueve muro o grupo seleccionado
- **Arrastrar esquina**: Redimensiona muro rectangular
- **Click derecho**: Elimina muro (si no estás en modo polígono)
- **Rueda del ratón**: Zoom in/out

## Pincel

- **+**: Aumenta tamaño del pincel
- **-**: Reduce tamaño del pincel

## Otras Funciones

- **G**: Guardar muros manualmente
- **F**: Fusionar muros seleccionados (solo rectangulares)
- **C**: Exportar muros a CSV
- **DEL**: Eliminar muro(s) seleccionado(s)
- **0**: Resetear zoom a 1:1
- **H / [?]**: Mostrar/ocultar ventana de ayuda

## Panel Lateral

- Selecciona mapa desde el panel izquierdo
- Expande/cierra secciones por categoría

## Ventana de Ayuda

- Click en el botón **[?]** arriba a la derecha o presiona **H** para ver todos los atajos y funciones

---

**Nota:**

- El modo polígono permite crear muros de cualquier forma (triángulos, hexágonos, etc.)
- El arrastre y multi-selección funcionan tanto para muros rectangulares como poligonales
- La fusión solo está disponible para muros rectangulares
- El zoom máximo es 1.0 (escala real del juego)
- El editor guarda automáticamente cada cierto tiempo si hay cambios
